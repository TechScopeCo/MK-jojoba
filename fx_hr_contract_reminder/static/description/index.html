HR Contract Reminder

This Odoo module provides functionality to send reminders for expiring HR contracts. It allows you to configure the number of days before contract expiration to send the reminder, and the HR manager who will receive the notifications.

Key features:

  • Automatically send email reminders to the HR manager and employees for contracts that are about to expire.
  • Configure the number of days before expiration to send the reminder.
  • Select the HR manager user who will receive the notifications.
  • Customizable email templates for the reminders.

Settings Configuration

In the "Contract Reminder" section, you can set the following:

  • Contract Expiration Reminder: Set the number of days before the contract expiration date when the reminder should be sent.
  • HR Manager: Select the user who will be designated as the HR Manager to receive notifications/reminders about expiring contracts.

Outgoing Mail Server Configuration

Go to the "Technical" settings, expand the "Email" section, and select "Outgoing Mail Servers". Here, you can enter the details of your SMTP server, such as the server address, port, authentication credentials, and other settings required for Odoo to send emails successfully.

Email Reminder for HR Manager

This template lists the employees whose contracts are about to expire, along with their contract expiration dates. The HR manager can then follow up with these employees to review their contracts.

Email Reminder for Employee

This template is for the employee whose contract are about to expire, along with their contract expiration dates.
